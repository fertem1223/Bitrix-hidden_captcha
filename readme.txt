Модуль защиты от спама. Работает с использованием скрытого поля.
Как установить ?
!. Распаковать модуль в папку модулей
2. Установить средствами Битрикса
3. В настройках модуля "Защита от спама" включаем галочку "Использовать капчу на формах сайта"
4. На формы сайта подкидываем поле <input type="text" name="testing_humanity" class="testing_humanity" value="" required>

Особенности
1. Не пропускает даже тех ботов которые заполняют только поля required
2. Работает без использования внешних сервисов
3. Быстро устанавливается на формы Битрикса
4. Работает не только с формами но и с комментариями сделанными через блоги.
